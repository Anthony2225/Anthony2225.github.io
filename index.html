title:CentOS系统下的ss配置

安装ShadowSocks服务端

这里如果提示 pip 安装失败,,要重装 Python环境

配置安装必须环境3

  • 编译器和相关工具

下载 编译 安装Python

***有两个Python版本 看自己喜好***

  • Python2.7.4

  • Python3.6.3

  • 从共享库中删除符号以减少内存占用

安装/升级pip

二、创建配置文件

yum -y install python-setuptools && easy_install pip
pip install shadowsocks

touch /etc/shadowsocks.json

vim /etc/shadowsocks.json

{
“server”: “0.0.0.0”,
“port_password”: {
“端口1”: “密码1”,
“端口2”: “密码2”,
“端口3”: “密码3”
},
“local_address”: “127.0.0.1”,
“local_port”:1080,
“timeout”: 300,
“method”: “aes-256-cfb”
}

上边的三个端口代表你创建了三个可以连接到ss 的账户,,如果需要多的就再加上,如果至于要一个,就只保留一个就行了

ssserver -c /etc/shadowsocks.json -d start

ssserver -c /etc/shadowsocks.json -d stop

vi /etc/rc.local

/usr/local/bin/ ssserver -c /etc/shadowsocks.json -d start

复制之后 按Esc退出编辑模式 然后输入 :wq 回车退出 vim

vi /etc/init.d/shadowsocks

#!/bin/sh

chkconfig: 2345 90 10

description: Start or stop the Shadowsocks server

#

BEGIN INIT INFO

Provides: Shadowsocks

Required-Start: $network $syslog

Required-Stop: $network

Default-Start: 2 3 4 5

Default-Stop: 0 1 6

Description: Start or stop the Shadowsocks server

END INIT INFO

Author: xju qing0991@163.com

name=shadowsocks
BIN=/usr/bin/ssserver
conf=/etc/shadowsocks.json
start(){
$BIN -c $conf -d start
RETVAL=$?
if [ “$RETVAL” = “0” ]; then
echo “$name start success”
else
echo “$name start failed”
fi
}
stop(){
pid=ps -ef | grep -v grep | grep -v ps | grep -i "${BIN}" | awk '{print $2}'
if [ ! -z $pid ]; then
$BIN -c $conf -d stop
RETVAL=$?
if [ “$RETVAL” = “0” ]; then
echo “$name stop success”
else
echo “$name stop failed”
fi
else
echo “$name is not running”
RETVAL=1
fi
}
status(){
pid=ps -ef | grep -v grep | grep -v ps | grep -i "${BIN}" | awk '{print $2}'
if [ -z $pid ]; then
echo “$name is not running”
RETVAL=1
else
echo “$name is running with PID $pid”
RETVAL=0
fi
}
case “$1” in
‘start’)
start
;;
‘stop’)
stop
;;
‘status’)
status
;;
‘restart’)
stop
start
RETVAL=$?
;;
*)
echo “Usage: $0 { start | stop | restart | status }”
RETVAL=1
;;
esac
exit $RETVAL

复制之后 按Esc退出编辑模式 然后输入 :wq 回车退出 vim

chmod a+x /etc/init.d/shadowsocks

chkconfig --add shadowsocks
chkconfig --list shadowsocks
#  应该会提示  shadowsocks    0:off1:off2:on3:on4:on5:on6:off

# 启动
service shadowsocks start
# 停止
service shadowsocks stop
# 状态
service shadowsocks status

Instagram 保存图片教程

网页版

  • 首先打开Instagram网站 https://www.instagram.com (有墙自己搬梯子)
  • 另开一个标签页打开 https://www.inssave.com/
  • 在 Instagram 网页 邮件想下载的图片选择 复制链接地址
  • 在 issave 网页把复制的图片地址复制到输入框 点击 提交
  • 点击图片下方红色字体的 下载 就OK了

安卓版

  • 首先 下载一个APP FastSave for Instagram (地址: https://play.google.com/store/apps/details?id=photo.video.instasaveapp&hl=en )
  • 打开FastSave 打开Switch 开关 ,然后点击Switch开关下边的 Open Instagram 按钮,打开Instagram,
  • 在Instagram里面遇到你想下载的图片,右上角有一个三个点的按钮,选择复制网址 然后切换到FastSace,这时候你会发现你的相册文件夹,多了一个*InstaSave相册,打开就是你刚刚保存的图片了.enjoy~
